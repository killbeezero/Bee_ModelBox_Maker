# 專案交接文件：ModelBoxLabel

## 專案概述

ModelBoxLabel 是一個跨平台標籤合成應用，用戶可以輸入產品信息，搜索相關圖片，並在畫布上合成標籤，最後導出為 PNG 圖片。目前已實現 Web 版本（React + Vite）和 macOS 版本（Swift + AppKit）。

## 功能需求

### 1. 用戶輸入
- **產品分類**：文字輸入欄位，用戶輸入產品類別。
- **機體型號**：文字輸入欄位，用戶輸入產品型號。
- **字型選擇**：下拉選單，提供 Arial、Helvetica、Times New Roman、Courier New 四種字型選項。

### 2. 圖片搜索
- 使用 Google Custom Search API 進行圖片搜索。
- 需要配置 API Key 和 Search Engine ID（目前使用：AIzaSyBqr2CCrrDoDh0IU2iZJ4Dsb_QYBcs_5G4 和 92788f79efe234975）。
- 搜索查詢基於產品分類和機體型號組合。
- 返回最多 10 張大尺寸圖片。
- 如果輸入不完整，顯示提示訊息。

### 3. 畫布合成
- **固定背景**：自動加載 `label.png` 作為畫布背景。
- **自動文字添加**：
  - 產品分類文字：位置 (10, 10)，字型根據用戶選擇，字體大小 24px。
  - 機體型號文字：位置 (10, 50)，字型根據用戶選擇，字體大小 24px。
- **手動圖片添加**：用戶可以從搜索結果的圖片列表中點擊圖片，將其添加到畫布上（位置 100,100，縮放 0.5）。
- 畫布尺寸：1583 x 661 像素，背景白色。

### 4. 導出功能
- 將畫布內容導出為 PNG 圖片。
- 自動保存或下載，文件名為 `label.png`。

## 技術架構

專案支持多平台實現，以下為各版本的技術棧：

### Web 版本
- **前端框架**：React 19.1.1
- **語言**：TypeScript 5.9.3
- **構建工具**：Vite 7.1.7
- **畫布操作**：Fabric.js 5.5.2
- **HTTP 請求**：Axios 1.12.2
- **開發工具**：ESLint 9.36.0, TypeScript ESLint 8.45.0

### macOS 版本
- **框架**：AppKit
- **語言**：Swift
- **開發工具**：Xcode
- **畫布操作**：Core Graphics / NSImage
- **HTTP 請求**：URLSession

## 安裝和運行

### Web 版本
1. **安裝依賴**：
   ```
   npm install
   ```

2. **開發模式**：
   ```
   npm run dev
   ```
   啟動開發伺服器，預設運行在 http://localhost:5173/

3. **構建生產版本**：
   ```
   npm run build
   ```

4. **預覽生產版本**：
   ```
   npm run preview
   ```

5. **代碼檢查**：
   ```
   npm run lint
   ```

### macOS 版本
1. 在 Xcode 中打開 MacApp 目錄。
2. 運行專案（Cmd + R）。

## API 使用說明

- **Google Custom Search API**：
  - URL：`https://www.googleapis.com/customsearch/v1`
  - 參數：
    - `key`：API Key（目前使用：AIzaSyBqr2CCrrDoDh0IU2iZJ4Dsb_QYBcs_5G4）
    - `cx`：Custom Search Engine ID（目前使用：92788f79efe234975）
    - `q`：搜索查詢（產品分類 + 機體型號）
    - `searchType`：image
    - `num`：10
    - `imgSize`：large
